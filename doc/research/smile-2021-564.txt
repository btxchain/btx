Figure 7 main protocol
- Commit v_1 .. v_m, w_0 = A * y_0, and g before sampling c_0.
- First Fiat-Shamir surface includes (~t_0, t_1 .. t_{m+k}, t_{k+2m}, w~, m).
- z_0 = y_0 + c_0 * s.
- P_1 = NTT(-c_0 * pk_1 | ... | -c_0 * pk_n).
- x_1 = NTT(w_0 - A * z_0).
- Recurse with SM_j for j = 1 .. m-1.
- Derive gamma_m, build the final public x_{m+1} object, then y_m.
- Derive alpha_0 .. alpha_m, build psi / omega terms, commit t_{k+2m+1}, derive c.
- z = y + c * r, then apply rejection sampling and verify with Figure 9.

Figure 8 recursive sub-protocol
- For j = 1, gamma_1 is in M_q^(k*l).
- For j >= 2, gamma_j is in M_q^l.
- P_{j+1} is the challenge-compressed matrix from Eq. (21).
- x_{j+1} = P_{j+1}(v_{j+1} tensor ... tensor v_m).
- y_1 = v_1 o x_2 - sum_i x_{1,i} o gamma_{1,i}.
- y_j = v_j o x_{j+1} - x_j o gamma_j for j >= 2.
- Each x_{j+1} is committed as public proof state.

Figure 9 verifier
- Lines 01-02: norm checks on z_0 and z.
- Line 03: B_0 * z = w + c * t_0.
- Line 04: adjust t_{m+1} .. t_{m+k} by subtracting A * z_0.
- Line 05: f_j = <b_j, z> - c * t_j.
- Lines 06-09: derive gamma values and x_{m+1} from the public recursion state.
- Lines 10-12:
  F_1 = f_1 * f_{m+k+1} + c * sum_i gamma_{1,i} * f_{m+i}
  F_j = f_j * f_{m+k+j} + c * gamma_j * f_{m+k+j-1}
  F_m = c - sum_i x_{m+1,i} * f_i + gamma_{m,1} * f_{k+2m-1} - e_1 * sum_i gamma_{m,i+1}
- Line 13:
  alpha_0 * (sum_j F_j - c * f_{k+2m} - c^2 * h) + sum_i alpha_i * (f_i^2 + c * f_i) + f_{k+2m+1} = omega
- Lines 14-15: first d/l coefficients of h are zero.

Figure 10 rejection sampling
- Rej0 is for z_0.
- Rej1 is for z.
- Rej1 rejects immediately when <z, v> < 0.

Eq. (21) recursive matrix compression
- P_{j+1} is formed from challenge-weighted transposes of the l block-columns of P_j.
- BTX code implication: compression must use the full gamma^T * P row product, not a diagonal shortcut.

Eq. (23) final public matrix
- x_{m+1,i} for i < m comes from B^T * gamma_{m,i+1}.
- x_{m+1,m} comes from P_m^T * gamma_{m,1} + B^T * gamma_{m,m+1}.
- BTX code implication: the last public x term needs the transpose-shaped P_m relation.

Eq. (24) final residual
- y_m = sum_i v_i o x_{m+1,i} - x_m o gamma_{m,1} - e_1 o sum_i gamma_{m,i+1}.

Eq. (31) public first-round matrix
- P_1 uses the negative sign: NTT(-c_0 * pk_i).
- x_1 = NTT(w_0 - A * z_0).

Eq. (39) first Fiat-Shamir c_0 surface
- c_0 is sampled only after the first-round transcript already includes w = B_0 * y and m.
- BTX code implication: c_0 binding is incomplete if w or m are missing from the transcript.

Appendix E / Figure 17 CT notes
